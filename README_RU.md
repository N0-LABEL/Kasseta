# Дискорд-бот футбольных счётов (русский)

Этот бот реализует команды, аналогичные `footballnationbot`, и автоматически присоединяется к указанному голосовому каналу и удерживает соединение (возвращается при перемещении/кике).

## Возможности

- /help — Показать список команд
- /live [команда-или-лига] — Подписка на лайв-счёты для канала
- /live-stop [команда-или-лига] — Остановить лайв для указанной подписки
- /live-stop-all — Остановить все лайв-подписки для канала
- /live-list — Показать список текущих подписок
- /live-upcoming — Показать ближайшие матчи
- /live-now — Показать текущие лайв-матчи
- /league-table [лига] — Таблица лиги
- /league-streaks [лига] — Серии (стрики) по лиге

Примечание: интеграция с внешним футбольным API вынесена в модуль `bot/services/football_provider.py`. По умолчанию используются заглушки. Для полноценной работы укажите поставщика и ключ API.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` (можно взять за основу `.env.example`) и заполните переменные:
- DISCORD_TOKEN — токен вашего Discord приложения
- TARGET_GUILD_ID — (опционально) ID сервера, где будет работать бот
- TARGET_VOICE_CHANNEL_ID — ID голосового канала, куда бот должен заходить и удерживаться
- FOOTBALL_API_PROVIDER — (опционально) поставщик данных, напр. `apifootball`
- FOOTBALL_API_KEY — (опционально) ключ для футбольного API

3. Запустите бота:
```bash
python -m bot.main
```

## Хранение подписок
Подписки каналов сохраняются в `data/subscriptions.json` и подгружаются при старте.

## Важно о правах
Боту нужны права:
- Читать и писать в каналы, где вы вызываете команды
- Использовать слэш-команды
- Подключаться/говорить в целевом голосовом канале

## Замечания по API
Модуль `football_provider.py` содержит интерфейс к API. Реализация по умолчанию — заглушки. Подключите реальный API (например, API-Football) и реализуйте методы получения счётов/таблиц/статистики.
